表单标签：收集用户信息

表单的三个部分：表单域 表单控件（表单元素） 提示信息

表单域form

action 指定接收并处理表单数据的服务器程序的url地址

method 设置表单数据的提交方式 get或post

name 指定表单名称 区分同一个页面中的多个表单域

表单元素

input 输入

type 指定不同类型的控件

name 指定input的名称 区分不同的input

value 指定input的值 给后台用的 text的value会显示出来 可以提示 但会占行一般不用

checked 默认值 单选和复选按钮可以有默认值

placeholder 不占行默认值

maxlength 规定输入字段的字符的最大长度

label 标签--为input元素定义标注

label用于绑定一个表单元素 当点击label标签内的文本时 浏览器会自动将焦点转移到对应表单元素上

就是不用一定要点那个圆点挥着或者框 点图标或者文字就行了

label的for属性和input的id属性对应

label是用来包字或者图片啥的


select 下拉列表

select 选项option 默认选项selected

textarea 文本域--输入多行文本

属性 row显示的行数 cols每行中的字符数 一般不用 都用css
