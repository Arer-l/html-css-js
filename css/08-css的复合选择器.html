css的复合选择器

常用：后代 子 并集 伪类

后代选择器：可以选择父元素里的子元素 可以更加精确地选择标签

格式：外层标签写在前面 内层标签写在后面 中间用空格分割 内层标签是外层标签的后代 改的是后代的样式 不一定是儿子 孙子 反正在这个标签内部的就是后代 一层一层往下找就行 但不用一层一层写 没有歧义的话头尾层就够了

后代选择器中复合的元素可以是任意一种基础选择器 本身格式保留（标签 类 id）

子选择器：只能选择作为某元素的最近一级子元素---亲儿子元素

格式：元素1 > 元素2 { 样式声明 }

1是父亲 2是儿子

我是儿子

我是孙子

并集选择器：可以选择多组标签 并同时为他们定义相同的样式 常用于集体声明

格式：元素1,元素2 {样式声明} 约定的语法规范 喜欢竖着写 最后一个选择器后面不用加逗号

并集选择器中复合的元素可以是任意一种选择器（基础和复合的都可以） 本身格式保留（标签 类 id）

gil

twins

伪类选择器：向某些选择器添加特殊的效果（a:hover这种）

链接伪类

a:link 选择所有未被访问的链接

a:visited 选择所有已被访问的链接

a:hover 选择鼠标指针位于其上的链接

a:active 选择活动链接 点下去的瞬间

要按照LVHA的顺序写（love hate） 一般只用a或link和hover

链接指定样式要单独指定

一个link

:focus伪类选择器：用于选取获得焦点（光标）的元素（一般是表单元素之类的）

格式：input:focus{样式} 谁获得了光标（点击）谁就会变
